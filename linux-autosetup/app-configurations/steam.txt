Start Without Game Launcher (boot straight into game):
	DO=(%command%); "${DO[@]/%FalloutNVLauncher.exe/FalloutNV.exe}"

AMD GPU Optimizations:
	mesa_glthread=true (mileage varies)
	Enable ACO (command can be put in steam launch options):
		sudo vim /etc/environment
		RADV_PERFTEST=aco

Run GameMode:
	gamemoderun
Enable AMD Fidelity Super Resolution:
	WINE_FULLSCREEN_FSR=1

Add to end of environment variables: %command%
